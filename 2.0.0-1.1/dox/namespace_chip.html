Detailed Description

Defines support for asserting that the chip stack is locked by the current thread via the macro:

assertChipStackLockedByCurrentThread()

Makes use of the following preprocessor macros:

CHIP_STACK_LOCK_TRACKING_ENABLED - keeps track of who locks/unlocks the chip stack CHIP_STACK_LOCK_TRACKING_ERROR_FATAL - lock tracking errors will cause the chip stack to abort/die

template<class ConfigClass>
class GenericConfigurationManagerImpl : public chip::DeviceLayer::ConfigurationManager

Provides a generic implementation of ConfigurationManager features that works on multiple platforms.

This template contains implementations of select features from the ConfigurationManager abstract interface that are suitable for use on all platforms. It is intended to be inherited (directly or indirectly) by the ConfigurationManagerImpl class.

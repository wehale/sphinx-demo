Creates an iterator that may be used to obtain the list of labels associated with the given endpoint. In order to release the allocated memory, the Release() method must be called after the iteration is finished. Modifying the label during the iteration is currently not supported, and may yield unexpected behaviour.

Return values

  • An – instance of EndpointIterator on success

  • nullptr – if no iterator instances are available.

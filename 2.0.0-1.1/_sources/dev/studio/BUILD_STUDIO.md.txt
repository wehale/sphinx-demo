# Building a Matter Application in Simplicity Studio

Once you have your project configured in Simplicity Studio you can build the project by right-clicking the project title in the Project Explorer view and selecting `Build Project` as seen below.

![Build in Simplicity Studio](./images/build-in-simplicity-studio.jpg)

Other build methods are available. More information on building your project inside Simplicity Studio is provided in the [Simplicity Studio Users Guide](https://docs.silabs.com/simplicity-studio-5-users-guide/latest/ss-5-users-guide-building-and-flashing/building).

# SiWx917 SoC Factory Reset

1. Press the BTN0 on WSTK for about 6-7 seconds. The LED0 will flash 5 times.

2. You will get "Target Connection Lost" pop-up on Ozone, click **YES**.
   ![Silicon Labs - SiWx917 SoC Ozone Target Connection Lost](./images/SiWx917_SoC_TargetConnectionLost.png)

3. Select **Attach to Running Program** in the dropdown next to the **Power** button on the top-left hand corner of the page.
   ![Silicon Labs - SiWx917 SoC Ozone Target Connection Lost](./images/SiWx917SoC_AttachToRunningProgram.png)

# Debugging a Matter Application in Simplicity Studio

You can also run the Simplicity Studio IDE's debugger from the Project Explorer view. Right click the project and select `Debug As > Silicon Labs ARM Program`

![Debug in Simplicity Studio](./images/debug-simplicity-studio.jpg)

More information on debugging in Simplicity Studio is provided in the [Simplicity Studio User's Guide](https://docs.silabs.com/simplicity-studio-5-users-guide/5.5.0/ss-5-users-guide-testing-and-debugging/using-the-debugger).

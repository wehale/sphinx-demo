Public Members

System::Clock::Milliseconds32 ActiveIntervalMS

Interval at which the device is able to communicate with its parent when there are active chip exchanges in progress. Only meaningful when the device is acting as a sleepy end node.

System::Clock::Milliseconds32 IdleIntervalMS

Interval at which the device is able to communicate with its parent when there are NO active chip exchanges in progress. Only meaningful when the device is acting as a sleepy end node.
